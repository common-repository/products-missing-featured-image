== Description ==

This plugin shows a list of Woocommerce products that do not have a featured image assigned. It saves a lot of time to ensure that all of the products in your store look good with a proper featured image.

The list can be accessed from WP Admin Menu -> Tools -> Missing Featured Image

== Installation ==

1. Upload `products-missing-featured-image.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
